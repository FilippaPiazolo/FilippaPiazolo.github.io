This tool is an interactive visualisation of phenotyps (human and mouse) that have been associates with SNPs or other locations along the genome. The visualisation includes the annotation of each SNP and also the density distribution of all SNPs on the chromosomes. Using the density distribution clusters can be automatically defined by giving extra prerequisits, such as minimum cluster size and minimum number of included SNPs and a threshold value of the kde-score. The density between different datasets and experiments can also be visually compared to each other.

Here two files of datasets with each up to six different experiments can be uploaded. Also other configurations such as genome type or reference genome can be set here. On this page the configuration and parameter settings can be downloaded and uploaded in the next session.

Data Visualisationmore_vert

See SNP/genomic annotations.

Data Visualisationclose

Here the SNPs and other locations on the genome are visualised as colored lines on the chromosomes. Different experiments and datasets can be selected via checkboxes.

Density Visualisationmore_vert

View density distribution and clusters.

Density Visualisationclose

Using kernel density estimation with the Epachnikov kernel we can define the density distribution from the given SNPs. Througth sliders the parameters of the calculation can be changed. The denisty is than used to define clusters. For details of the cluster a link to the UCSC genome browser is included.

Density Comparisonmore_vert

Compare density distribution between datasets.

Density Comparisonclose

Here the density between two different dataset or experiments can be compared. The desnity of one dataset is green and the other one is red. When the density of both datasets on an area of the chromosome is similar it gets yellow.
